# Project scaffold — Little Acre M0 (board + day loop)

**Date:** 2026-07-07 · **Branch:** main (initial bootstrap import) · **Roadmap item:** M0 — Vertical Slice

## Goal

Stand up a clean, runnable Next.js project for Little Acre, mirroring the Disaster Co.
tooling/conventions but with a **Canvas2D** renderer, and port the `Farm Idle.dc.html`
prototype into a playable M0 slice (iso board + Day/Sleep/Night loop) that builds,
type-checks, lints, and has passing Vitest tests.

## Approach

- Mirror Disaster Co.'s config exactly (package.json scripts, tsconfig strict + `@/*`
  alias, eslint flat config, `.prettierrc`, husky + lint-staged, vitest `@` alias) —
  drop `pixi.js`; the renderer is the plain 2D canvas context.
- Separate pure logic (`lib/game`) from the Canvas renderer (`lib/renderer`) from the
  React shell (`components`), same discipline as the reference project.
- Port the prototype's `class Component` logic into a Zustand store + pure `tiles.ts`
  resolvers, and its `draw*` methods into a typed `BoardRenderer` with an imperative
  handle. Game.tsx wires pointer input → store actions → renderer fx, never
  re-rendering React per frame.

## What actually happened

Shipped the full M0 slice. All four gates clean: `pnpm lint`, `pnpm type-check`,
`pnpm test` (**20 tests**), `pnpm build`. Prod server smoke-tested (200, correct title).

**Ported from the prototype:**
- Isometric board renderer — tiles (grass/tilled/pond/rock/flower), 4-stage crops,
  wilt art, ponds (ripples + fish), rocks (ore/gem specks), flowers, sprinkler &
  scarecrow, particle fx (burst/splash/coin/floating text), tile pop/nudge/place-bounce
  animations, screen shake, ambient pollen, sun-glow, hover outline.
- Full Day → Sleep → Night → dawn loop with Energy spend/refill.
- Click tool: water · harvest (free) · clear wilted · fish · mine (coins + ~22% gem).
- Build tool: plant crops · place land · build structures, with cost/energy guards.
- HUD (coins/gems/day/bloom/energy + Sleep), hotbar (Click/Build/Store), build picker,
  Store modal (Shop/Boost/Guide + Rebloom), toasts, night overlay.
- Versioned localStorage save (`little-acre-v1`) + 10s autosave + tab-hide/unload persist.

**Stubbed / presentational (intentional for M0):**
- Store **Shop** and **Boost** tabs are display-only — no purchase economy yet.
- **Rebloom** is a simple ×1.6 Bloom + 8-gem reset (real prestige math is open).
- Fishing/mining use the prototype's simple RNG payouts; no tool gating.
- No DOM hover-tooltip box (the canvas hover outline stands in) — fine for touch.
- Offline progress is a stub — the night only advances on explicit Sleep (matches the
  prototype; the idle-vs-active question is open, see GDD §8).
